# UX Wireframes â€” Zoning Intelligence Platform

## Design Philosophy

**Reference**: Apple.com, Square.com, Linear.app

### Core Principles

1. **Minimal** â€” Every element earns its place
2. **Calm** â€” No visual noise, no anxiety-inducing UI
3. **Spacious** â€” Generous whitespace, breathing room
4. **Typography-first** â€” Hierarchy through type, not decoration
5. **Clear hierarchy** â€” Instant understanding of what's important
6. **Trust-building** â€” Professional users must trust this instantly

### Visual Language

- **Colors**: Near-white backgrounds (#fafafa), soft grays, one accent (blue #0071e3)
- **Typography**: SF Pro / system fonts, light weights for headings, readable body
- **Borders**: Subtle (0.5px), low contrast, only where needed
- **Shadows**: Soft, barely visible, used sparingly
- **Corners**: Consistent large radius (16-24px for cards, 8-12px for inputs)
- **Spacing**: 8px grid, generous padding (24-48px)
- **Icons**: Line icons, consistent stroke width, never decorative
